## EQUINOX

A fast-paced, top-down space shooter game where players pilot the spacecraft captained by Nova, battling through waves of enemies across different locations in the galaxy.

### Key Features

- Top-down space shooter gameplay
- Wave-based enemy combat system
- Player-controlled spacecraft with shooting mechanics
- Multiple galaxy locations
- Score tracking and progression

### Tech Stack & Architecture

- **Language:** Java
- **Framework:** Java Swing for GUI and game rendering
- **Architecture:** Object-oriented design with game loop pattern

### Project Information

Academic game development project focusing on object-oriented programming principles and Java Swing GUI development.
